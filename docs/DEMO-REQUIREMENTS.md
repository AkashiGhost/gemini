# Demo Requirements

This is the short version of what the judges need to see in the main demo.

## Hard Requirements

- Public YouTube or Vimeo link
- Under 4 minutes
- Real working software, not mockups
- Clear problem and clear solution
- English voiceover or English subtitles

## What The Main Demo Must Prove

1. InnerPlay breaks the text-box pattern
2. The product is actually live and playable
3. Gemini is driving the interaction in real time
4. The system runs on Google Cloud
5. There is real technical depth behind the experience

## Recommended Structure

1. Hook and problem: `0:00-0:30`
   - Show a player with eyes closed
   - State the core idea fast
   - Frame the problem: most AI storytelling is still screen-first

2. Real experience: `0:30-1:30`
   - Show one real Gemini Live exchange
   - Let the judges hear the live voice
   - Show the UI only enough to prove it is real software

3. Why it is different: `1:30-2:15`
   - Show creator flow
   - Show player profile or memory/personalization
   - Show that stories can be created and published, not only played

4. Architecture: `2:15-3:00`
   - Show the architecture diagram
   - Explain browser mic, Gemini Live, Cloud Run, live tools, and layered audio
   - Keep it readable, not dense

5. Close: `3:00-3:35`
   - Restate the product in one sentence
   - End on the strongest product line
   - Show the URL

## Must-Include Shots

- Player wearing headphones with eyes closed
- Real InnerPlay browser session
- Real Gemini Live audio clip with visible product UI
- One creator pipeline clip
- One architecture diagram clip
- Final product URL frame

## Must-Include Talking Points

- "InnerPlay is a voice-only AI storytelling game designed to be played with your eyes closed."
- Built with Gemini Live / Google GenAI SDK
- Hosted on Google Cloud Run
- Real-time tool-driven story control
- Three-layer audio experience: voice, ambience, score

## What To Keep Out Of The Main Demo

- Terminal windows
- Cloud Run console
- Raw logs
- Admin dashboards
- Long mouse movement
- Any mention of old or non-Google providers

## Separate Artifact Rule

The Cloud Run proof must be separate from the main demo video. Do not spend demo time showing terminal output or infrastructure proof.
